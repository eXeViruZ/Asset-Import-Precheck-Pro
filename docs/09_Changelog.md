# 09 — Changelog

---

## 1.0.0 — 2026

**Initial release.**

### Features
- 5-step guided import wizard
- AssetRegistry.bin integration (registry mode + filesystem fallback)
- 4-severity system: Safe / Warning / High Risk / Blocked
- 5 precheck rules: Path Safety, Engine Name Collision, Fallback Note, Name Conflict, Missing Dependencies
- 3 import modes: Import All / Skip Conflicting / Import Safe Only
- Dependency viewer: Flat list + Tree view
- Severity filter buttons on precheck step
- Async scan + import (background thread, throbber UI)
- Last source path persistence via editor config
- Export report to clipboard and CSV
- Tools menu entry (Tools → Asset Import Precheck Pro)
- Level Editor toolbar button — persistent AIPP button in the main editor toolbar for one-click access
- Full copyright headers on all source files
